Sport Buddy

Sport Buddy is a web app that allows you to find sports sessions near you. You can sign up and then search for sports sessions. You can also add your own sessions and others can sign up for them. The app is written in Ruby on Rails and uses the Mapbox API to display sports sessions on a map.
